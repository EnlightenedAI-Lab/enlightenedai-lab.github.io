02 //What We Evaluate

[E-01] We evaluate alignment as a dynamic property of reasoning systems, not as a stated objective.

Our work focuses on how alignment behaves under pressure: recursive self-reference, increasing model scale, and prolonged reasoning trajectories. These conditions are where failure modes emerge.

# Evaluation Strategy: Dynamic Stress Testing def evaluate_under_pressure(model: Model): drift = model.run_trajectory(EvaluationContext.RECURSIVE_REF) if drift > SAFTEY_MARGIN: return FailureMode.DETECTED

The lexicon is not descriptive documentation. It is a measurement instrument.
